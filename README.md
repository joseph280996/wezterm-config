# Wezterm config

This is where all of my configuration for Wezterm.
I'm only including the keymaps that I would want to use in Wezterm to avoid much conflict with Neovim.
All the keymaps can be found in [keymaps.lua](./config/keymaps.lua) and configuration can be found in [config.lua](./config/init.lua).

I'm also using tabline.wez, which is a Wezterm plugin that allows modification to the tabbar, which adopts the configuration setup philosophy similar to that of tabline.nvim
